# Best Hikes Around Ithaca
This project creates maps for the Best Hikes Around Ithaca book. 

## Project Overview:
The Best Hikes Around Ithaca book present 50 hikes and walking trails in and around Ithaca, NY.

## Built With:
Python, ArcPy
